How to make a site using Hugo, Github Actions and pages

March 20, 2021 One-minute read

This site was build with hugo, Github Actions, Markdown pages and a drop of love. Here is a couple of links I used:

  • hugo is a static site generator, also see a quick start guide
  • Github Actions is a CI/CD building blocks for Github
  • Github Pages is a free hosting for static websites
  • You can setup hugo with Github Actions and Pages using this link
  • Posts can be edited as Markdown documents, syntax cheat sheet is here
  • To be loved you should be nice.

Also, I replace --minify flag in my Actions file from the guide with -D, since the old command couldn’t generate new pages (UPD: -D == --buildDrafts, so you should not mark posts as drafts and everything will be OK with the default configuration)

Here is the CLI commands for hugo (also see cheat.sh):

1
2
3
4

hugo new site blog              # create directory 'blog' with site skeleton
hugo new posts/my-first-post.md # create first post blog
hugo server -D                  # start a local server with hot reloading
hugo -D                         # build site
